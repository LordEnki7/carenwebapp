## 🚀 One-Command Setup

```bash
# Extract and setup (choose one)
tar -xzf caren_complete_mobile_navigation_ready_*.tar.gz && cd caren_* && npm install
# OR
unzip caren_complete_mobile_navigation_ready_*.zip && cd caren_* && npm install

# Setup environment
cp .env.example .env
# Edit .env with your API keys and database URL

# Initialize database
npm run db:push

# Start development
npm run dev
```
