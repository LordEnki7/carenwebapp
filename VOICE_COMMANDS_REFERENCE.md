# C.A.R.E.N. Voice Commands Reference

## Complete List of Available Voice Commands

### Emergency Commands (Critical Priority)
**"CAREN Emergency"** or **"Emergency"**
- Activates emergency mode with immediate recording and family notification
- Auto-starts incident documentation
- Notifies emergency contacts
- Example: Say "CAREN Emergency" or just "Emergency"

**"CAREN Help"** or **"Help me"**
- Requests emergency assistance and starts incident documentation
- Activates high-priority recording mode
- Sends help request to emergency contacts
- Example: Say "CAREN Help" or "Help me"

### Recording Commands (High Priority)
**"Record Start"**
- Begins recording incident documentation immediately
- Activates video/audio capture
- Creates incident file with timestamp
- Example: Say "Record Start" or "Start recording"

**"Record Stop"**
- Stops current recording and saves incident documentation
- Finalizes incident file
- Confirms save completion
- Example: Say "Record Stop" or "Stop recording"

### Navigation Commands (Medium Priority)
**"Navigate Home"**
- Returns to main dashboard
- Shows overview and quick actions
- Example: Say "Navigate Home" or "Go home"

**"Navigate Rights"**
- Displays legal rights for current location
- Shows state-specific constitutional protections
- Provides legal guidance
- Example: Say "Navigate Rights" or "Show my rights"

**"Navigate Settings"**
- Opens application settings and preferences
- Access to device configuration
- Account management options
- Example: Say "Navigate Settings" or "Open settings"

### Information Commands (Low Priority)
**"Show Help"**
- Displays this voice commands reference guide
- Opens complete voice commands page
- Provides hands-free usage guidance
- Example: Say "Show Help" or "Voice commands"

**"Show Commands"**
- Alternative to "Show Help"
- Opens voice commands guide
- Lists all available commands
- Example: Say "Show Commands"

## Emergency Keywords (Auto-Trigger)
These words automatically activate emergency mode without prefixes:
- **"emergency"**
- **"help me"** 
- **"police"**
- **"pulled over"**
- **"traffic stop"**

## Voice Recognition Tips

### ✅ STABILITY IMPROVEMENTS (Updated July 6, 2025)

### Best Practices
- Speak clearly and at normal volume
- Wait for confirmation after each command
- Commands work in any order ("Emergency CAREN" = "CAREN Emergency")
- Voice recognition maintains stable connection automatically
- System now recovers gracefully from temporary interruptions

### Command Structure
- **Emergency**: Use "CAREN" prefix or emergency keywords
- **Recording**: Use "Record" prefix
- **Navigation**: Use "Navigate" prefix  
- **Information**: Use "Show" prefix

### Enhanced Stability Features
- **Automatic Error Recovery**: System restarts intelligently after network issues
- **Circuit Breaker Protection**: Prevents rapid cycling that caused instability
- **Smart Timeout Management**: Optimized restart delays for consistent performance

- No wake word needed - just speak any command
- Voice feedback confirms every action
- Real-time transcript shows what CAREN heard
- Commands processed with 95%+ accuracy

- Voice confirmation of command recognition

## Integration with CAREN Features
- **GPS-Aware**: Commands adapt to your current location

## Command Prioritization
1. **Critical**: Emergency activation commands
2. **High**: Recording and incident documentation
3. **Medium**: Navigation and settings access
4. **Low**: Information and help commands

Emergency commands always take priority and interrupt other operations.
