SUBMISSION INSTRUCTIONS

Copyright Office Submission:


  • Online Filing (Recommended):

- Go to copyright.gov
- Create account and use eCO system
- Upload digital files of source code and screenshots
- Pay $65 fee online
- Receive confirmation within 24 hours

  • Mail Submission:

- Print all documents
- Include source code printouts
- Mail to: U.S. Copyright Office, Library of Congress, 101 Independence Ave. SE, Washington, DC 20559-6000
- Include $125 check

Trademark Office Submission:


  • TEAS Online System:

- Go to uspto.gov
- Use Trademark Electronic Application System (TEAS)
- Pay fees online ($350 per class per mark)
- Upload specimen images

Patent Office Submission:


  • EFS-Web System:

- Go to uspto.gov
- Use Electronic Filing System
- Requires detailed technical drawings and claims
- Recommended to use patent attorney
