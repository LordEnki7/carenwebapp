# C.A.R.E.N. Auth Module

## Overview
This module handles auth-related functionality for the C.A.R.E.N. platform.

## Structure
```
auth/
├── frontend/          # React components, hooks, and services
├── backend/           # Express controllers, services, and routes
├── shared/            # Shared types and schemas
├── tests/             # Unit and integration tests
├── package.json       # Module dependencies
├── module.config.json # Module configuration
└── README.md          # This file
```

## Installation
```bash
npm install @caren/auth
```

## Usage
```typescript
import { /* exports */ } from '@caren/auth';
```
