# C.A.R.E.N. Emergency Module

## Overview
This module handles emergency-related functionality for the C.A.R.E.N. platform.

## Structure
```
emergency/
├── frontend/          # React components, hooks, and services
├── backend/           # Express controllers, services, and routes
├── shared/            # Shared types and schemas
├── tests/             # Unit and integration tests
├── package.json       # Module dependencies
├── module.config.json # Module configuration
└── README.md          # This file
```
